# Projeto Wanda - Materiais Didáticos

Repositório destinado ao armazenamento e organização dos materiais educacionais desenvolvidos para o **Projeto Wanda**, iniciativa voltada ao ensino de lógica de programação e pensamento computacional para estudantes do Ensino Fundamental (8º e 9º ano).

O Projeto Wanda tem como objetivo introduzir conceitos fundamentais da programação de forma acessível, lúdica e contextualizada, utilizando exemplos do cotidiano, atividades práticas e recursos visuais que facilitem a aprendizagem.

Os materiais aqui disponibilizados são utilizados na elaboração de aulas, oficinas, videoaulas e demais ações educacionais do projeto.

* Desenvolver o raciocínio lógico e computacional.
* Introduzir conceitos básicos de programação.
* Estimular a resolução de problemas.
* Promover o interesse pela área de tecnologia.
* Disponibilizar recursos didáticos para apoio às atividades do projeto.

As gravações de videoaulas também podem ser acessadas através do canal no YouTube ProjetoWanda
disponíveis clicando no [link](https://youtube.com/@projetowanda?si=t7TPK8m0KbN2zadq).
